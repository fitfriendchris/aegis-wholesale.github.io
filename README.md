# Aegis Capital | Off-Market Real Estate Deals

Live site: https://aegis-wholesale.github.io

## Pages
- `index.html` — Main landing page (seller + buyer capture)
- `deal-analyzer.html` — Property analysis tool (ARV, MAO, pipeline)
- `email-templates.html` — Copy-paste email templates

## Tech Stack
- HTML5 + Tailwind CSS CDN
- LocalStorage for data persistence
- Zero backend required

## Deploy
Push to GitHub Pages:
```bash
git remote add origin https://github.com/yourusername/aegis-wholesale.github.io.git
git push -u origin main
```
